# Audio Transcription and Text Analysis Pipeline

This project implements an end-to-end pipeline for converting audio files into text transcripts and performing basic text processing and analysis using Python.

## Project Overview
- Converts audio files into text transcripts
- Organizes transcripts into structured formats (CSV/text)
- Performs basic text cleaning and analysis
- Implemented using Jupyter Notebooks

## Folder Structure
- `my audio files/` – Raw audio input files
- `transcriptions/` – Generated transcript outputs
- `texts/` – Processed text data
- `Assignment 2.ipynb` – Main implementation notebook
- `transcriptions.xlsx` – Consolidated transcript results

## Technologies Used
- Python
- Jupyter Notebook
- Speech-to-text processing
- Basic NLP techniques

## Notes
This project was developed as part of a university coursework assignment and demonstrates practical handling of audio data, transcription workflows, and text analysis.
